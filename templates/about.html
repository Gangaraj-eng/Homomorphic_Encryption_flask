Website working

This website is built using python Flask framework.The frontend part is completely html, css and javascript and backend is complete python. The database used is Azure CosmosDB API for mongodb and is hosted and running on Azure cloud. Login credentials entered by youu are first encrypted using Homomorphic Encryption and then stored in database. So, Your details will be safe in cloud.

When you upload and excel sheet, the data is encrypted and sent to cloud and run by the cloud functions which are also hosted on Azure cloud platform.The cloud will perform computations on encrypted data and then returns the result which we then decrypt and display to you.

The excel sheet format currently supported should be shown below.There should be 4 columns and first column corresponds to id or name of student and next three columns corresponds to marks in three subjects. The marks should be integers as we currently Fully Homomorphic encryption supports only integers.
